#### 表格
表格使用[|]来分割不同的单元格，使用[-]来分隔表头和其他行
- [:-] 将表头及单元格内容左对齐
- [-:] 将表头及单元格内容右对齐
- [:-:] 将表头及单元格内容居中

#### 图片

<img src="head.jfif" width=60 height=60  />

- Ctrl + Shift + I（Typora）
- 对齐
```
![img](URL#pic_center)
```
```HTML
<div align=center>
  <img src=URL />
</div>
```
- 大小
```
![img](URL =60x60)
```
```HTML
<img src=URL width=60 height=60  />
```
